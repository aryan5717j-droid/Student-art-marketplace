
Student Art Hub - Ready-to-run static website
--------------------------------------------
Files included:
- index.html
- art.html
- books.html
- sketches.html
- notes.html
- styles.css
- script.js

How to use:
1. Download and unzip the package.
2. Open index.html in a browser (double-click) to run locally.
3. To host online: upload the extracted files to GitHub Pages, Netlify, or any static hosting.

This is a demo static site (no backend). JS alerts simulate buying/messages.
